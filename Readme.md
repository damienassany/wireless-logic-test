# Configuration

| Tool    | Version |
| ------- | ------- |
| Node.js | 16.14.2 |
| Yarn    | 1.22.10 |

# Setup instructions

- Clone the repo
- run `yarn` to install dependencies
- run `yarn start` to run the script

# Commands

| Command    | Description        |
| ---------- | ------------------ |
| yarn start | Run the script     |
| yarn test  | Run the unit tests |
